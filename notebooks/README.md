# Notebooks

Use `Early_CVD_Prediction_ML_Lab_Supervisor_Demo.ipynb` for the final ML Lab demonstration. It follows the report sections, loads the generated project artifacts, shows the data audit, compares models, presents ablation/sensitivity studies, and runs an explainable individual prediction.

The original `Heart_Disease_Prediction_03_SmallSet_Feat_Eng_NeuralNet_ModDef_Training-Copy1.ipynb` is preserved as historical exploratory work. It is useful for showing the starting point, but the final reported results come from the production pipeline in `src/`.
